## Firebase Sync Skill

This skill enables the Antigravity agent to synchronize analyzed patent data, expert insights, and trend statistics with a Firebase Firestore database. This data can then be visualized on a dashboard.

### Usage:
The agent will use this skill to update the Firebase dashboard after performing patent analysis or generating new insights.

### Example:
"Update the Firebase dashboard with the latest patent analysis results."
