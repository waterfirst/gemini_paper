## Patent Search Skill

This skill allows the Antigravity agent to search for patent information from various databases.

### Usage:
The agent will use this skill when prompted to find patents related to specific technologies, companies, or within certain date ranges.

### Example:
"Find patents related to HBM4 from Samsung in the last 48 hours."
